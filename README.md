# KidsConstructionMemory

A fun and colorful memory game for kids! Match pairs of construction vehicles in a playful, cartoon-style environment. This project is a personal initiative to practice web development during my training.

## Demo

finishedmemorygame.netlify.app

## Features

- Cartoon-style construction vehicles (bulldozers, cranes, diggers, etc.)  
- Interactive memory game mechanics  
- Simple and colorful interface for children  
- Tracks score and moves  

## How to Play

1. Click on a card to reveal the vehicle.  
2. Find its matching pair.  
3. Matched cards stay visible.  
4. Try to match all pairs in the fewest moves!  

## Technologies Used

- HTML, CSS (with CSS variables for colors)  
- JavaScript (game logic)  
- Optional: images in cartoon style for the vehicles
